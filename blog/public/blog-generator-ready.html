The Deployment Strategy

We solved the GitHub Pages deployment problem with a simple but elegant approach:

Safe Drafting

1# Write freely - never goes live
2vim blog/drafts/my-ideas.md
3git push  # ‚Üê No deployment triggered

Intentional Publishing

1# Publish when ready
2git mv blog/drafts/my-ideas.md blog/published/002-my-ideas.md
3git push  # ‚Üê Deployment triggered automatically

Key insight: Publishing requires explicit file movement. No accidents, clear intent, perfect git history.
